## Description

This interactive program introduces users to basic cryptography by implementing the Caesar Cipher—a type of substitution cipher where each letter in the plaintext is shifted a fixed number of positions down the alphabet. The tool prompts the user to choose between encoding (encrypting) or decoding (decrypting) a message, enter their text, and specify a shift key (number) to instantly calculate and display the output.

## Features

- **Double Functionality:** Seamlessly switch between encrypting (encoding) and decrypting (decoding) messages.
- **Custom Shift Key:** Input any shift number to determine how far the letters are moved down the alphabet.
- **Smart Formatting:** Preserves spaces, numbers, and special symbols exactly as they are while only shifting alphabetic letters.
- **Simple Console Interface:** Straightforward, text-driven prompts for ease of use in the terminal.
- **Beginner-Friendly Cryptography:** Well-commented, readable code demonstrating list indexing, loops, and string manipulation.

## Technologies Used

- Python

## How to Run

1. Download or clone this repository.
2. Open the project in PyCharm or any Python IDE.
3. Run `caesar-cipher.py`.
4. Choose whether to encrypt or decrypt, enter your secret message and shift number, and watch your text transform!

## Objective

Securely scramble and unscramble secret messages using one of the oldest historical cipher methods.
